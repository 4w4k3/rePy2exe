# rePy2exe

Reverse Engineering Tool for py2exe applications. 


### Prerequisites

* cmake
* git
* python2.7

### Cloning
```
git clone https://github.com/4w4k3/rePy2exe.git
```

## Running
```
python rePy2exe.py
```
or
```
python2.7 rePy2exe.py
```
